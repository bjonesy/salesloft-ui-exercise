# SalesLoft UI Exercise

A fluid & responsive page layout using HTML, SCSS, and JS that is based on the Gmail's web interface.

## Features:

* [HMR](https://webpack.js.org/concepts/hot-module-replacement/)
* [SASS](http://sass-lang.com/)

## Setup

- `npm install`
- `npm start`
- Visit [http://localhost:3000/](http://localhost:3000/)

## Toggle Menu

The toggle menu has three options.

- All
- Read
- None

<img src="http://www.brandonsj.me/wp-content/uploads/2017/08/Screen-Shot-2017-08-13-at-5.18.11-PM.png" width="850">

### All

Select all emails at once.

### Read

Select all emails that have been read.

### None

Deselect all selected emails.

## Email list view toggle

- Increase Text
- Expand View
- Simple View

<img src="http://www.brandonsj.me/wp-content/uploads/2017/08/Screen-Shot-2017-08-13-at-5.26.06-PM.png" width="850">

### Increase Text

Increase the test of the emails.

### Expand View

Expand the default view of the emails to display the full message.

### Simple View

Display a minimal view of the email list.

## Toolbar Email Functions

<img src="http://www.brandonsj.me/wp-content/uploads/2017/08/Screen-Shot-2017-08-13-at-5.24.33-PM.png">

- Delete Emails
- Favorite Emails
- Spam Emails

### Delete Emails

When an email is selected pressing the trash icon will move the email to the deleted tab in the sidebar menu

### Favorite Emails

When an email is selected pressing the disk icon will move the email to the favorite tab in the sidebar menu

### Spam Emails

When an email is selected pressing the chat bubble with the exclamation mark icon will move the email to the spam tab in the sidebar menu.
